## What is useState and why do you use it?

### I'll give you the basics and very common example of using useEffet that most of the React developer's used.

`OLD way using "this.state" in a class base component`

```js
import React, { Component } from "react"
import "./App.css"
//Lifecycle method
class App extends Component {
  constructor(props) {
    super(props)
    this.state = { learn: false, num: 1 }
    this.handleClick = this.handleClick.bind(this)
  }

  handleClick() {
    this.setState(() => ({ learn: !this.state.learn }))
  }
  componentDidMount() {
    //side effects after initial load
    console.log("mounted")
  }
  componentDidUpdate() {
    //side effects after reload cause by state change
    console.log("updated")
  }

  render() {
    return (
      <div className="App">
        <h1 onClick={this.handleClick}>Learn React Using "Class components"</h1>
        <p>Learn React {this.state.learn ? "NEXT MONTH" : "NEXT YEAR"}?</p>
      </div>
    )
  }
}
export default App
```
